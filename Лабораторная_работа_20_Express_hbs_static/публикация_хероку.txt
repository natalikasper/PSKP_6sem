сайт хероку - https://dashboard.heroku.com/
1.установить heroku cli(lci)
2.в 20-01 заменить "app.listen" на "var server=app.listen(process.env.PORT || app.get('port'));"
3.выполнить логирование через установленное приложение heroku
	3.1.открываем cmd
	3.2.чтобы проверить установку - heroku --version
	3.3.heroku login (будет написано нажать что-либо для регистрации, откроется браузер и вып.регистр)
4.в баузере добавить приложение
5.слонировать репозиторий
	5.1.dit init
	5.2.heroku git:remote <название приложения в heroku>
	(дальше по картинкам в папке)
6.выполнить команды гита для deploy
	(дальше по картинкам в папке)
